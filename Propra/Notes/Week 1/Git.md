# Git
- A version control software

![[Pasted image 20220911170924.png]]

- `git init` initializes a git repository.
- `git add [filename]` adds file to the staging area.
- `git add *` adds all files to the staging area.
- `git commit -m "message"` commits changes to **HEAD** with a message, usually something useful.
- `git push` sends the changes to the remote repository
	- Use `git push origin [branch]` to send changes to a specific branch.
